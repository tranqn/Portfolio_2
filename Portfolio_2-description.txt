# Portfolio Design Specification

## Global Components

### CTA Button
Reusable call-to-action button used across multiple sections.
- On light background: black text, white background, black border, black box-shadow
- On dark background: white text, black background, white border, white box-shadow
- Hover (both variants): white text, orange background, translate to box-shadow ending coordinates

### Animation Technique
All animations below are **images revealed with CSS `clip-path`**, except the Skills "Skill Set" circle animation (see below).

### Arrow Animation
Arrow image revealed with clip-path. Used in: Colleagues Thoughts section, Contact section.

### Header Navigation Hover Animations
Each nav item has a unique hover animation (all clip-path image reveals):
- "About Me": circle image over text
- "Skills": underline image under text
- "Projects": cross-out image over text
- "Contact": underline image under text

### Section Headline Animations
Each section has a unique animation on its headline (all clip-path image reveals, except Skills):
- About Me: underscore image beneath "About Me" headline
- Projects: underscore image beneath "Projects" headline
- Skills: circle image over "Skill Set" headline — starts at -90deg rotation and scaled down, animates to 0deg rotation and original size, also revealed with clip-path

## Sections

### 1. Hero Section (Above the Fold)

**Layout:** 100vh, single background image covering entire section (header included within 100vh, sharing the same background).

**Header:**
- Left: website logo
- Center: navigation menu (About Me, Skills, Projects, Contact) — each item has a unique hover animation (see Global Components)
- Right: language toggle switch (German / English, default: German)
- All header elements equally sized/spaced
- Mobile: logo + hamburger menu icon only

**Hamburger Menu (mobile only):**
- Hamburger icon transforms to X icon on open
- Navigation menu (About Me, Skills, Projects, Contact)
- Language toggle switch
- Social media icons (GitHub, LinkedIn, Email)

**Hero Content:**

Desktop — two main columns:

Left column:
- Social media icons (GitHub, LinkedIn, Email) with white circular borders
- Aligned to bottom of section

Right column, top row (two columns):
- Left:
  - "Hello World" button (top)
    - Hover: text changes to "I'm Quoc Nam Tran", blue background expands from center to expanded box size, overlay image fades in aligned to the right rotating 180 degrees with slight bounce-back effect
  - "FRONTEND" text (bottom)
    - Hover: each letter gets uppercase and yellow color
- Right: Polaroid-style image
  - Default: grayscale, text ":)"
  - Hover: color, text ":D"

Right column, bottom row:
- Top: "DEVELOPER" text
- Bottom: "Get in Touch" button (CTA)

Mobile — single column, stacked:
- "Hello World" button
- "FRONTEND" text
- "DEVELOPER" text
- "Get in Touch" button (CTA)
- Polaroid image
